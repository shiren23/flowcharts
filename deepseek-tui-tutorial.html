1. DeepSeek TUI 是什么？

DeepSeek TUI 是一个完全运行在终端里的编程 AI 智能体。它让 DeepSeek V4 模型直接访问你的工作区：读写文件、运行命令、搜索网页、管理 Git —— 全部通过键盘驱动完成。

核心特性：100 万 token 上下文 · 思考模式流式推理 · MCP 协议 · 子智能体并行 · 中文界面支持

简单来说：它就像是活在终端里的"Cursor"或"Claude Code"，但驱动模型是 DeepSeek。

2. 前置准备：获取 API Key

DeepSeek TUI 需要 DeepSeek API Key 才能运行。按以下步骤获取：

1

打开 DeepSeek 开放平台

浏览器访问 https://platform.deepseek.com，注册并登录账号。

2

创建 API Key

进入「API Keys」页面，点击「创建 API Key」。复制生成的 Key（以 sk- 开头），只会显示一次。

3

充值（如需要）

新注册账号通常有免费额度。如果没有，在「充值」页面充值即可（最低 10 元）。

提示：DeepSeek V4 Flash 价格非常便宜（输入 $0.14/百万 token），日常编程辅助一天花费不到 1 元。

3. 安装 DeepSeek TUI

Mac 上推荐以下两种方式，选其中一种即可：

方式一：Homebrew 安装（推荐）

如果你已经安装了 Homebrew，这是最简单的方式：

Terminal 
brew tap Hmbown/deepseek-tui
brew install deepseek-tui
没有 Homebrew？先安装它：打开终端，粘贴下面的命令执行：
Terminal 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

方式二：npm 安装

如果你电脑上有 Node.js（前端开发者通常已有）：

Terminal 
npm install -g deepseek-tui
中国大陆用户加速：使用镜像源安装更快：
Terminal 
npm install -g deepseek-tui --registry=https://registry.npmmirror.com

方式三：直接下载

不想装任何包管理器？直接下载二进制文件：

1

打开 Releases 页面

访问 https://github.com/Hmbown/DeepSeek-TUI/releases

2

选择对应版本

Apple Silicon (M1/M2/M3/M4) 下载 deepseek-tui-*-aarch64-apple-darwin.tar.gz

Intel Mac 下载 deepseek-tui-*-x86_64-apple-darwin.tar.gz

3

解压并移动到 PATH

Terminal 
# 解压（替换为实际文件名）
tar xzf deepseek-tui-*-aarch64-apple-darwin.tar.gz

# 移动到用户可执行目录
sudo mv deepseek /usr/local/bin/
sudo mv deepseek-tui /usr/local/bin/

验证安装

安装完成后，运行以下命令确认：

Terminal 
deepseek --version

看到版本号输出即安装成功。

4. 首次配置

方式一：交互式配置（推荐）

直接启动 DeepSeek TUI，首次运行会自动引导你输入 API Key：

Terminal 
deepseek

按提示粘贴你的 API Key，回车即可。Key 会保存在 ~/.deepseek/config.toml，以后无需重复输入。

方式二：命令行配置

Terminal 
deepseek auth set --provider deepseek

按提示输入 Key 即可。

方式三：环境变量

如果你不想保存到文件，也可以每次通过环境变量传入：

Terminal 
export DEEPSEEK_API_KEY="sk-你的密钥"
deepseek

想永久生效，把 export 那行加到 ~/.zshrc 文件末尾：

Terminal 
echo 'export DEEPSEEK_API_KEY="sk-你的密钥"' >> ~/.zshrc
source ~/.zshrc

切换为中文界面

DeepSeek TUI 原生支持中文。如果首次启动是英文，在 TUI 内：

  1. 在输入框输入 /config，按 Enter 打开配置面板
  2. 选择 Edit locale，输入 zh-Hans，按 Enter

或直接编辑配置文件：

Terminal — config.toml 
# ~/.deepseek/config.toml
[tui]
locale = "zh-Hans"

验证配置是否正确

Terminal 
deepseek doctor

这条命令会检查你的安装状态、API Key 配置和网络连通性。全部显示绿色即配置成功。

第三步：在打开的终端窗口中输入命令

终端会自动定位到该文件夹目录，直接输入 deepseek 并回车即可启动。

没有「在新终端窗口打开」选项？这是 macOS 的内置服务功能。如果看不到，请确保你的 Mac 系统版本为 macOS 12 或更高。也可以在系统设置 → 隐私与安全性 → 辅助功能中检查终端权限。

一次性提问（不进入 TUI）

Terminal 
deepseek "解释一下这个函数的作用"

指定模型

Terminal 
# 使用 Flash 模型（快速、便宜）
deepseek --model deepseek-v4-flash "快速总结这个文件"

# 使用 Pro 模型（更强推理）
deepseek --model deepseek-v4-pro "帮我重构这个模块"

# 让 TUI 自动选择模型（推荐）
deepseek --model auto

会话管理

Terminal 
# 列出已保存的会话
deepseek sessions

# 恢复最近一次会话
deepseek resume --last

# 按会话 ID 恢复
deepseek resume <SESSION_ID>

在 TUI 内部附加文件

在输入框中使用 @文件路径 将文件内容发送给 AI：

TUI 输入框 
@src/main.py 帮我找出这段代码里的 bug

7. 三种交互模式

模式说明适用场景
Plan 🔍 只读调查模式。AI 只探索和制定计划，不做任何修改 了解代码库、分析问题
Agent 🤖 默认模式。AI 可以执行操作，但每步需要你确认 日常编程、修改代码
YOLO ⚡ 自动批准模式。AI 自主执行所有操作，无需确认 可信项目、批量重构

在 TUI 中按 Tab 键可在模式之间切换。

注意：YOLO 模式下 AI 会自动执行命令和修改文件，请确保在可信工作区中使用。

8. 常见问题

Q：提示"command not found: deepseek"

安装后终端找不到命令，通常是因为安装路径没加入 PATH。尝试重新打开终端窗口，或执行：

Terminal 
source ~/.zshrc

Q：API Key 怎么更换？

Terminal 
# 清除旧 Key
deepseek auth clear --provider deepseek

# 重新设置
deepseek auth set --provider deepseek

Q：如何更新到最新版本？

Terminal 
# 使用内置更新命令
deepseek update

# 或使用包管理器更新
brew upgrade deepseek-tui    # Homebrew
npm update -g deepseek-tui   # npm

Q：Mac 上终端推荐用哪个？

推荐以下终端应用（都免费）：

  • iTerm2 — 最流行的 Mac 终端替代品，功能强大
  • Warp — 现代 AI 终端，界面美观
  • Terminal.app — 系统自带，开箱即用

Q：收费吗？

DeepSeek TUI 本身完全免费开源。但你调用 DeepSeek API 会产生费用，V4 Flash 模型非常便宜，日常使用每天不到 1 元。新注册账号通常有免费额度。

Q：与 Claude Code 相比有什么区别？

两者都是终端里的 AI 编程智能体，核心区别是驱动模型不同。DeepSeek TUI 使用 DeepSeek V4 模型，价格更低；Claude Code 使用 Anthropic 的 Claude 模型。功能上各有特色，可以根据需求选择或搭配使用。
